Overview

A complete rebuild of my personal portfolio — cleaner structure, stronger storytelling, and a more intentional developer identity.

The Problem

My previous portfolio didn’t reflect my growth. It lacked structure, didn’t tell my story well, and didn’t showcase my projects with enough depth or clarity.

The Solution

I designed and built Portfolio v2 from the ground up: a modular, semantic, multi‑page site that highlights my journey, my work, and my voice as a developer.

Features

  • 8‑page structure (Home, About, Projects, Deep Dives, Blog, Contact)
  • Consistent navigation and layout across all pages
  • Semantic HTML5 for clarity and accessibility
  • Project deep‑dive pages with Problem → Solution → Tech Stack
  • Blog section for devlogs and insights
  • Clean folder structure with assets separated

Tech Stack

  • HTML5 for structure
  • CSS (optional: Tailwind or custom styles)
  • JavaScript for interactivity
  • Version control with Git

What I Learned

This project helped me refine my storytelling, improve my semantic HTML structure, and think like both a developer and a product designer. It also strengthened my ability to present my work clearly and professionally.

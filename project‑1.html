Overview

A full delivery workflow system with order management, driver assignment, and live tracking.

The Problem

Small delivery businesses struggle with manual coordination, slow communication, and lack of real‑time visibility.

The Solution

I built a complete delivery management system that automates order flow, assigns drivers intelligently, and provides live tracking for customers and admins.

Features

  • Customer order placement
  • Driver dashboard with task queue
  • Admin panel for full oversight
  • Live map tracking
  • Status notifications

Tech Stack

  • Frontend: HTML, CSS, JavaScript, React
  • Backend: Node.js, Express
  • Database: MongoDB
  • APIs: Google Maps API

What I Learned

This project strengthened my understanding of real‑time systems, API integration, and designing multi‑role dashboards.

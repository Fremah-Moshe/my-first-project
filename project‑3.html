Overview

A clean, minimal platform for writing, editing, and publishing blog posts.

The Problem

Most blogging tools are bloated or overly complex for someone who just wants to write and publish quickly.

The Solution

I built a lightweight blogging system with a simple editor, a posts dashboard, and a clean reading experience.

Features

  • Rich text editor
  • Post creation and editing
  • Tag system
  • Public blog feed
  • Responsive layout

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express
  • Database: MongoDB

What I Learned

This project helped me understand CRUD operations deeply and build clean, reusable UI components.

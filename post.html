Seeing Computers as More Than Just Devices

In the first week, I learned that a computer isn’t just a machine I use every day—it’s a system built on layers of logic. Understanding how the CPU, memory, and storage work together helped me appreciate how instructions move through the system. Before this, I never thought about what actually happens when I open a browser or run a program, but now I can picture the flow of data and how the hardware responds to commands.

This new perspective made computers feel less mysterious. Instead of thinking of them as black boxes, I started seeing them as predictable tools that follow clear rules. That shift alone made me more confident about diving deeper into programming and web development.

Learning the Structure and Meaning Behind HTML

HTML was another big part of Week 1, and what stood out to me most was how much meaning is built into the language. Tags like <header>, <main>, and <footer> aren’t just for layout—they describe the purpose of each part of the page. Once I understood that, writing HTML felt less like typing code and more like organizing ideas in a logical way.

I also practiced writing clean sentences using proper HTML syntax, and it made me realize how important structure is for both humans and browsers. Whether it’s a paragraph, a list, or a section, each element has a job. Learning this early helped me build better habits as I continue exploring web development.

Experimenting With HTML5 Media

One of the most exciting parts of the week was experimenting with HTML5 media elements. Adding a video, an audio clip, and even an embedded webpage through an <iframe> showed me how flexible HTML can be. It was satisfying to see how a simple tag could bring interactive content into a page without needing any extra software.

These tools made the web feel more dynamic. Instead of thinking of a webpage as just text and images, I started seeing it as a space where different types of media can work together. It made me appreciate how modern websites create richer experiences using just a few well‑chosen elements.

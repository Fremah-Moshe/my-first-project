Overview

A focused task manager designed to keep daily work clear, organized, and actually doable.

The Problem

Most todo apps either feel too basic or too overwhelming, making it hard to stay consistent and focused.

The Solution

I built a lightweight productivity app that balances simplicity with just enough structure: categories, priorities, and gentle reminders.

Features

  • Create, edit, and delete tasks
  • Task categories (Work, Personal, Learning, etc.)
  • Priority levels and due dates
  • Filter and search tasks
  • Responsive layout for desktop and mobile

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Optional: React for components and state
  • Storage: LocalStorage (or simple backend API)

What I Learned

This project sharpened my skills in state management, UX for everyday tools, and keeping interfaces clean but powerful.
